此脚本依赖AutohotKey。你可以使用Ctrl+Alt+空格来打开ChatGPT
如果想开机自启可以进入startup文件夹（通过下载的bat）
然后新建一个txt。后缀改成bat右键编辑。输入 cd 你的脚本存放路径  (回车)  start 脚本名称(例如：ChatGPT.ahk)
